Panduan

Install dalam 3 langkah.

1

Download APK

Tap tombol download di halaman ini. File Aniku.apk tersimpan otomatis ke folder Downloads.

2

Aktifkan izin install

Buka Pengaturan → Keamanan lalu aktifkan "Install dari sumber tidak dikenal". Ini hanya perlu dilakukan sekali.

3

Buka & install APK

Buka file manager, tap Aniku.apk, lalu tap Install. Selesai — nikmati anime gratis!
